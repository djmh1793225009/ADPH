# ADPH: AIGC提示词语料库构建在建筑插画风格中的运用

本项目是数字媒体艺术专业的毕业设计作品，旨在解决 AIGC（人工智能生成内容）在建筑设计领域应用时存在的随机性高、文本控制力不足、难以维持结构稳定性等痛点。通过跨学科技术整合，构建了一套针对建筑行业的高效辅助工作流与可视化交互系统。

## 🌟 项目亮点

- **结构化语料库**：深入解构建筑专业语言，建立自然语言与建筑形式语言的精确映射，涵盖地域文化、物理材质、光影环境等维度。
- **精确空间约束**：利用 Canny（边缘检测）和 Depth（深度估算）模型锁定建筑几何拓扑，实现"同构异质"的稳定风格迁移。
- **本地化工作流**：基于 ComfyUI/Stable Diffusion 节点式框架，优化显存分配，实现高效、可控的批量生产。
- **交互式前端展示**：开发 Web 端交互系统，通过滑动对比、数据联动监视面板直观展示 AIGC 辅助设计的逻辑与成果。
- **专业采纳建议**：为每种风格撰写了深度结合建筑设计与插画设计双重视角的专业采纳建议，指导实际应用场景选型。

## 🚀 技术路线

1.  **数据预处理**：对采集的提示词进行清洗降噪，剔除无效修饰词，从 `SOP_v1.0` 逐步迭代至 `SOP_v3.0` 结构化表格。
2.  **几何底模构建**：选取现代综合体作为建筑原型，提取 Canny 轮廓线与 Depth 深度图。
3.  **节点工作流搭建**：在本地部署 Stable Diffusion 环境，接入特定建筑材质的 LoRA 微调权重。
4.  **风格迁移实验**：在空间约束下，针对传统地域、现代材质、幻想概念进行多维度渲染测试，覆盖通用建筑、内部空间、远景空间、立面设计、材质质感、平面图纸、特殊视角等 8 大类别。
5.  **采纳建议优化**：为每种风格撰写专业的采纳建议，融合建筑设计与插画设计双重视角，指导用户根据项目需求精准选型。
6.  **可视化开发**：利用React 构建前端页面，实现同构异质切片的动态拼接展示。

## 🛠️ 如何使用

### 1. 提示词库
查阅 `doc/SOP_v3.0/SOP_v3.0.xlsx`（最新版）或 `doc/SOP_v2.0/SOP_v2.0.xlsx`，可直接获取针对建筑风格、材质、光影优化的正向/反向提示词及采纳建议。

### 2. 交互网站
进入 `nodejs/` 目录：
```bash
npm install
npm run dev
```
启动后可在浏览器查看建筑风格流变影像及实时参数映射。

### 3. 图像生成
`tmp/ti2i.py` 包含图像生成控制脚本，可配合本地 Stable Diffusion 环境进行批量风格迁移。

## 📝 结论与意义

本研究不仅产出了一套高质量的建筑插画风格图集，更重要的是跑通了"数据清洗-工作流搭建-空间约束渲染-前端交互展示"的全链路闭环。它证明了在数字化工具时代，创作者可以通过参数框架锁定空间逻辑，将 AI 转化为精准的渲染执行工具。
